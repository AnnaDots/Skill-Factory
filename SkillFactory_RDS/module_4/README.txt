Проект 4 "Компьютер говорит нет"

Цель проекта: создание модели банковского скоринга. Предсказание вероятности дефолта клиента на основании информации о нём с использованием алгоритма логистической регрессии (задача классификации).

Содержание проекта:
Описание датасета
EDA
   -Числовые параметры
   -Бинарные параметры
   -Категориальные параметры
Обучение наивной модели
Генерация новых параметров
Поиск гиперпараметров
Создание submission файла

На данном проекте были отработаны навыки EDA (очистка и преобразование признаков), оценка важности признаков, подготовка данных для обучения модели, обучение модели логистической регрессии, расчёт целевой переменной (допустит ли клиент дефолт) и оценка работы модели различными метриками (accuracy, precision, recall, F1-score, кривая ROC-AUC и матрица ошибок), генерация новых признаков и подбор гиперпараметров с помощью Grid Search.

Работа со следующими библиотеками:
-pandas
-numpy
-matplotlib
-seaborn
-datetime
-sklearn
